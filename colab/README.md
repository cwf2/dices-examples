# Colab notebooks

This is a collection of DICES-client tutorial notebooks that are designed to be used in Google Colab. Each of these notebooks assumes that it's the only one visible to the user, and that packages have to be re-installed every time the notebook is run.

### Quickstart &nbsp;&nbsp; [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/cwf2/dices-examples/blob/main/colab/Quickstart.ipynb)
 - A set of copy-paste-able examples demonstrating basic tasks with DICES

### Dead Flavians &nbsp;&nbsp; [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/cwf2/dices-examples/blob/main/colab/Dead%20Flavians.ipynb)
 - Who speaks to dead people in Flavian epics? Demonstrates working with character attributes, tabular data, and waffle charts.

### NLP &nbsp;&nbsp; [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/cwf2/dices-examples/blob/main/colab/NLP.ipynb)
 - Download the text of speeches in Silius Italicus. Parse with a language model to generate lemma and part of speech tags for every word. A basic walkthrough of the NLP workflow, tested on Colab free tier.
 - **Note**: this notebook depends on third-party tools and is subject to change.
